## Getting Started

```bash
# Clone this repository
git clone https://github.com/flashbots/suapp-examples.git
```

---

## Compile the examples

Install [Foundry](https://getfoundry.sh/):

```bash
curl -L https://foundry.paradigm.xyz | bash
foundryup
```

Install dependencies:

```bash
forge install
```

Compile:

```bash
forge build
```
